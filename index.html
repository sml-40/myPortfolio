About Me

My name is Selina Lavery. I am a Junior Software Engineer with experience in the fields of marketing, finance, and training.

I live in Stonehaven, a beautiful seaside town in North-East Scotland with my husband, two kids and two cats.

In 2019, I completed a writing course with The Writer’s College which resulted in three of my short stories being published with online magazines and one featuring on a Podcast. I began research to create my own website.

This led to my next passion. Coding.

After learning HTML and CSS on Codecademy, I decided to explore my interest in coding further.

I have recently graduated from a software engineering immersive programme with General Assembly where I learned a wide variety of programming and computer science concepts.

I have developed complete web applications using JavaScript, React, Ruby on Rails, and the MERN Stack.

GENERAL ASSEMBLY PROJECT 1

Tic-Tac-Toe

Build Time: 6 days | Project type: Solo

This project allowed me to put into practice my learnings of HTML, CSS, and JavaScript. The challenge was to recreate the classic tic-tac-toe game in six days. I used HTML and CSS to design the 3x3 grid and user interface, while the functionality was implemented using Vanilla JavaScript.

GENERAL ASSEMBLY PROJECT 3

Fitness Tracker

Build Time: 8 days | Project type: Group

GENERAL ASSEMBLY PROJECT 4

Build Time: 5 days | Project type: Solo

For my final project, I had the option to select a tech stack to create an application with two models and full CRUD on one model.
I built an application called ‘Budget Plan Pal’ using full stack Ruby on Rails. My project illustrated my ability in applying Rails associations, authenticating with Devise, asset pipeline and PostgreSQL. Styled with SASS.
